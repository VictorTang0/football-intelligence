# 🧠 球队特性性格特征 (Team Tag System) 独立 Skill 模块

本 Skill 旨在通过预测结论与真实赛果/赛场数据的动态比对，独立发展出一套衡量球队特性的 Tag 体系，并将该体系作为自适应权重微调核心挂钩在模型预测工作流中。

---

## 📌 核心运行准则

### 1. Tag 周期与衰减机制 (Decay)
*   **层数升级 (Level Up)**：当赛果触发某标签条件时，标签层数 `level + 1`，并将不匹配计数器 `consecutive_mismatches` 重置为 `0`。
*   **不匹配衰减 (Decay)**：若完赛后评估数据不符合已有标签，其 `consecutive_mismatches + 1`。若连续 **3** 场均不符合该标签，其层数 `level - 1`；当 `level` 归零时，该标签彻底被移除。
*   **对立冲突机制 (Exclusion)**：标签配置包含 `opposite_tags`。一旦触发对立标签，原对立标签立即从该队档案中清空，以防止因子权重打架。

### 2. 预测权重微调规则 (Weight Adjustment)
*   **起效门槛**：只有**层数 $\ge 2$ 层**的标签才会在预测时用于微调因子的参考权重。$1$ 层的标签不进行参考。
*   **调权算法 (Option A)**：
    *   获取球队活跃 Tags ($\text{level} \ge 2$) 映射的决策因子 ID。
    *   临时上调该因子权重：
        $$\text{Weight}_{\text{temp}} = \text{Weight}_{\text{original}} \times (1.0 + 0.15 \times \text{level})$$
    *   将所有 41 项决策因子权重等比重归一化（使总和重新精确等于 $100\%$）。
*   **界面展示**：
    *   **终极预测区域卡片**：仅展示主/客队层数最高（相同时展示更新时间最新）的**最重要的 Tag 徽章**。
    *   **情报详情面板**：在 Intelligence 面板的顶部独立网格，罗列出该队所持有的**所有 Tags 及其当前 Level 层数**。
    *   **因子面板**：受 Tag 调权的因子表格行及进度条会高亮发光，并渲染 `🏷️ 调权` 提示徽章。

## 📋 20 个标签的定义与判定标准

1.  **🎭 明星演员 (Star Actor)**：赛前胜赔 $< 1.50$ 的强队爆冷平局或输球。
    *   *对立*：稳如老狗、科学战车。
    *   *影响*：调增 `"09"` (近期状态)、`"22"` (压力稳定性)、`"40"` (冷门)。
2.  **🔄 剧本反转 (Plot Twist)**：比赛战果与预测结论彻底相反。
    *   *对立*：科学战车、稳如老狗。
    *   *影响*：调增 `"40"` (冷门)、`"22"` (压力稳定性)。
3.  **🚀 顺风狂飙 (Windfall Storm)**：取得 3 球及以上的绝对大胜。
    *   *对立*：闪退大客车、只雷不雨。
    *   *影响*：调增 `"07"` (前锋把握)、`"19"` (边路传中)、`"17"` (定位球进攻)。
4.  **🐑 欺软怕硬 (Bully the Weak)**：输给强队且对手赔率 $<1.70$。
    *   *对立*：逆转专家、德比狂战士。
    *   *影响*：调增 `"01"` (整体实力)、`"22"` (压力稳定性)。
5.  **🧱 铜墙铁壁 (Iron Wall)**：零封对手 (Conceded Goals = 0)。
    *   *对立*：灌球高手。
    *   *影响*：调增 `"06"` (后防稳定性)、`"08"` (门将状态)、`"18"` (定位球防守)。
6.  **💣 灌球高手 (Goal Packer)**：单场轰入 3 球及以上。
    *   *对立*：铜墙铁壁、闪退大客车。
    *   *影响*：调增 `"07"` (前锋把握)、`"17"` (定位球进攻)。
7.  **🤝 平局大师 (Draw Master)**：完赛结果为平局。
    *   *影响*：调增 `"06"` (后防稳定性)、`"12"` (整体磨合)。
8.  **🏳️ 无心恋战 (Heartless Fighter)**：完败零进球。
    *   *对立*：抢分狂魔、德比狂战士。
    *   *影响*：调增 `"25"` (形势压力)、`"29"` (士气影响)。
9.  **⚙️ 科学战车 (Scientific Tank)**：完赛比分与预测的 `most_likely_score` 完全一致。
    *   *对立*：明星演员、剧本反转。
    *   *影响*：调增 `"01"` (整体实力)、`"04"` (战术匹配)、`"12"` (整体磨合)。
10. **⏳ 虎头蛇尾 (Tiger's Head)**：主队先进球但未能赢下比赛。
    *   *对立*：续命狂人、逆转专家。
    *   *影响*：调增 `"10"` (赛程休息)、`"13"` (替补深度)。
11. **🪓 伐木大队 (Lumberjack Crew)**：赛前或赛中呈现高黄牌数。
    *   *影响*：调增 `"24"` (红黄牌停赛)、`"36"` (裁判尺度)。
12. **🏠 主场狂魔 (Home Monster)**：主场出战并赢球。
    *   *影响*：调增 `"35"` (球迷主场氛围)。
13. **⚔️ 德比狂战士 (Derby Berserker)**：在联赛德比战中赢下盘口。
    *   *对立*：无心恋战。
    *   *影响*：调增 `"22"` (压力稳定性)、`"29"` (士气影响)。
14. **📈 抢分狂魔 (Points Scavenger)**：预测命中且赢下抢分大战。
    *   *对立*：无心恋战。
    *   *影响*：调增 `"25"` (形势压力)、`"22"` (压力稳定性)。
15. **⏱️ 续命狂人 (Life Extender)**：险胜 (如 2-1, 1-2) 并在最后时段建功。
    *   *对立*：虎头蛇尾。
    *   *影响*：调增 `"21"` (落后追分)、`"22"` (压力稳定性)。
16. **⚡ 只雷不雨 (Thunder No Rain)**：得球 $\le 1$ 且未赢球的低效热门队。
    *   *对立*：顺风狂飙、灌球高手。
    *   *影响*：调增 `"05"` (中场控制)、`"14"` (出球抗压)。
17. **🔄 逆转专家 (Comeback Expert)**：落后盘面下最终成功逆转赢球。
    *   *对立*：欺软怕硬、虎头蛇尾。
    *   *影响*：调增 `"21"` (落后追分)、`"11"` (临场调整)。
18. **🦠 病毒受害者 (Virus Victim)**：国际比赛日后或密集赛程下爆冷输球。
    *   *影响*：调增 `"10"` (赛程休息)、`"30"` (旅行时差)。
19. **🛡️ 闪退大客车 (Blitz & Retreat)**：进球后摆大巴，依靠 1-0 最终赢球。
    *   *对立*：顺风狂飙、灌球高手。
    *   *影响*：调增 `"20"` (领先控场)、`"06"` (后防稳定性)。
20. **🏆 杯赛狂人 (Cup Maniac)**：在单场淘汰赛或杯赛场景中赢盘。
    *   *影响*：调增 `"22"` (压力稳定性)、`"25"` (形势压力)。

## 💻 挂钩与调用方式

*   **运行机制**：该系统的评估逻辑集成在 `evolve.py` 中。在赛后数据导入（Command C）时执行，更新 `data/team_tags.json`。
*   **模型应用**：在今日预测（Command A）或详细因子折线图生成时，通过 `getAdjustedWeights()` 自动调取活跃 Tags。
